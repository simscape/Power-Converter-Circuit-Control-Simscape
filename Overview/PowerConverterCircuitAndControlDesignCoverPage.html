Overview

In this project, you learn how to design power converter circuits and their control systems using Simscape. The project includes workflows for designing LLC resonant converters and grid-forming converter controllers. You will learn how to estimate resonant tank parameters, select converter components, tune compensator controllers, and design grid-forming inverter control strategies.

Get Started

Double-click on the PowerConverterCircuitControlSimscape PRJ file (in the root folder) to setup the project and get started.

Components

The project utilizes library components available in Simscape™ and Simscape Electrical™ libraries. There are custom Simscape components used in this project for power converter circuits and controllers. All such custom components, their documentation, source code, and test harness can be viewed by navigating to the Components folder in the project root directory. The combined library (ConverterLibrary) is in the Components/Utilities folder.

Click Component Documentation to view the full component list with documentation and test harness links.
