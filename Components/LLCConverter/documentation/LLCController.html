LLC Voltage Controller Block

The LLC Resonant Converter Controller block provides voltage regulation for LLC resonant converters. The block supports four controller architectures: 2P2Z (two-pole, two-zero) compensators and PI controllers, available in both continuous and discrete forms. You can configure flexible gains for the voltage-controlled oscillator (VCO) subsystem.
The block comprises three main subsystems:
  • Voltage controller
  • Voltage-Controlled Oscillator (VCO)
  • Gate Signal Generator

LLC Resonant Converter Power Circuit Internal Structure

  • Error Calculation: Calculates the error between the reference voltage (Ref) and the measured voltage (Meas).
  • Compensator: Implements the selected controller (2P2Z or PI, continuous or discrete). Converts the voltage error to a normalized output voltage.
  • VCO: Converts the compensator output to a frequency command for the gate signal.
  • Gate Signal Generator:Generates the PWM gate signals with dead-time.

Compensator Controller

This block implements a two-pole, two-zero compensator. One pole is placed at the origin to function as an integrator, with its gain adjustable to meet specific performance requirements. For applications with small input and output voltage variations, you can select the PI controller option to regulate the LLC converter.
  • Variants: S-domain or Z-domain 2P2Z, PI (continuous or discrete)
  • Parameters:Uses Kc, numSdomain, denSdomain, numZdomain, denZdomain, Kp, Ki, TsController.

Voltage Controlled Oscillator

You can configure the block to use either a constant VCO gain or two separate gains: one gain for frequencies below resonance (boost operation) and another gain for frequencies above resonance (buck operation).
  • Variants: Constant rate or two rate
  • Parameters:Uses vcoGain, vcoGainVec.

Gate Signal Generation

  • Function:Converts frequency command to gate signals, applies dead-time (Tdead), and handles signal formatting.
